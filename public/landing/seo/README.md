# Imágenes SEO para Formosa FMC

Este directorio contiene las imágenes utilizadas para SEO y redes sociales.

## Archivos necesarios

### Open Graph Image
- **Archivo**: `og-image.png`
- **Tamaño**: 1200x630px
- **Formato**: PNG optimizado
- **Uso**: Compartir en Facebook, LinkedIn, WhatsApp, etc.

### Twitter Card Image
- **Archivo**: `twitter-image.png`
- **Tamaño**: 1200x600px o 1200x1200px
- **Formato**: PNG optimizado
- **Uso**: Compartir en Twitter/X

## Contenido recomendado

Las imágenes deben incluir:
- Logo de Formosa FMC
- Texto: "Créditos para Motos"
- Texto: "Financiá hasta 36 meses"
- Texto: "Tasa fija o UVA • Aprobación en minutos"
- Colores de marca: Púrpura (#5b00ea), Azul (#3b82f6), Verde (#10b981)

## Generación de imágenes

Se han creado archivos SVG como base. Para generar las imágenes PNG finales:

1. Abrir los archivos SVG en un editor de imágenes (Illustrator, Inkscape, etc.)
2. Exportar como PNG con las dimensiones especificadas
3. Optimizar las imágenes con herramientas como TinyPNG o ImageOptim
4. Reemplazar los archivos SVG con los PNG optimizados

## Nota

Los archivos SVG actuales son placeholders. Se recomienda crear versiones PNG optimizadas con el logo real y diseño profesional.
